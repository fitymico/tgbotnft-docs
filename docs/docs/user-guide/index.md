# Руководство пользователя

Добро пожаловать в документацию **NFT Gift Bot** — бота для автоматической покупки NFT-подарков в Telegram.

---

## Быстрый старт

### 1. Откройте Service Bot

Перейдите в [Service Bot](https://t.me/l_servicebot) — это Ваш помощник для покупки подписки и управления ботом.

### 2. Купите подписку

Выберите подходящий тариф и оплатите через Telegram Stars:

| Тариф | Месяц | Год | Для кого |
|-------|-------|-----|----------|
| **SELF-HOST** | 109⭐ | 1090⭐ | Запуск на своём сервере |
| **HOSTING** | 169⭐ | 1690⭐ | Запуск на нашем сервере |
| **HOSTING-PRO** | 249⭐ | 2490⭐ | Выделенный VPS |

### 3. Настройте бота

После покупки следуйте инструкции для Вашего тарифа:

- **[SELF-HOST](./self-host)** — установка на своём сервере
- **[HOSTING](./hosting)** — настройка через Service Bot
- **[HOSTING-PRO](./hosting-pro)** — настройка и деплой админом

## Что такое NFT Gift Bot?

**NFT Gift Bot** — это инструмент для автоматической покупки Star Gift (NFT-подарков) в Telegram.

### Как это работает

1. Вы настраиваете бота и авторизуете свой Telegram-аккаунт
2. Бот подключается к центральному серверу и получает информацию о новых подарках
3. Когда появляется новый подарок, бот автоматически покупает его от Вашего имени
4. Вы получаете уведомление о покупке

### Зачем нужен собственный бот?

Ваш личный Telegram-бот служит интерфейсом для управления NFT Gift Bot:
- Отправляет уведомления о покупках
- Позволяет настраивать параметры
- Показывает статистику работы

## Часто задаваемые вопросы

### Безопасно ли это?

- Мы не имеем доступа к Вашим личным сообщениям
- Сессия используется только для покупки подарков
- Вы можете в любой момент отозвать доступ в настройках Telegram

### Что нужно для начала?

- Аккаунт Telegram с достаточным количеством Stars
- Bot Token от [@BotFather](https://t.me/BotFather)
- Для SELF-HOST: свой сервер с Linux

### Как быстро бот покупает подарки?

Скорость зависит от тарифа:
- **SELF-HOST** — зависит от производительности Вашего сервера
- **HOSTING** — стандартная скорость
- **HOSTING-PRO** — максимальная скорость

### Что делать, если бот не работает?

1. Проверьте статус в Service Bot (для HOSTING/HOSTING-PRO)
2. Посмотрите логи на наличие ошибок
3. Убедитесь, что сессия Telegram действительна
4. Свяжитесь с поддержкой через Service Bot
