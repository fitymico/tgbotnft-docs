# Для пользователя

Для создания уникального бота в первую очередь потребуется связаться с нашим ботом в Telegram.

## С чего начать

1. [Работа с Service Bot](./user-guide/service-bot) — знакомство с ботом и покупка тарифа
2. [Создание бота через BotFather](./user-guide/create-bot) — получение Bot Token
3. [Получение API ID и Hash](./user-guide/telegram-api) — для тарифа SELF-HOST

## Настройка по тарифу

- [Тарифы](./user-guide/tariffs) — сравнение тарифов и что входит в каждый
- [SELF-HOST](./user-guide/self-host) — установка бота на своём сервере
- [HOSTING](./user-guide/hosting) — управление ботом на нашем сервере
- [HOSTING-PRO](./user-guide/hosting-pro) — бот на выделенном VPS
