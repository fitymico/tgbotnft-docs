# Работа с Service Bot

Наш бот в Telegram: [Service Bot](https://t.me/)

1. При переходе в бота, нажмите кнопку `Старт`.
2. Выберите тариф, который хотите использовать, и купите за звезды Вашего аккаунта Telegram (после покупки Вашему аккаунту присвоится **ключ лицензии**).

### Доступные тарифы

| Тариф | Описание | Что входит |
|-------|----------|------------|
| **SELF-HOST** | Самостоятельная установка | Лицензионный ключ + документация по развёртыванию на своём сервере |
| **HOSTING** | Мы запускаем за Вас | Автоматический запуск бота на нашем сервере, управление через Service Bot |
| **HOSTING-PRO** | Выделенный VPS | Деплой на отдельном VPS, максимальная производительность |

> Подробнее о каждом тарифе и процессе настройки — в разделе [Тарифы](./tariffs).
