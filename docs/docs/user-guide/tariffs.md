# Тарифы

После оплаты в [Service Bot](https://t.me/) дальнейшие действия зависят от выбранного тарифа.

## SELF-HOST

**Для тех, кто хочет запускать бота на своём сервере.**

После оплаты Вы получаете:
- **Лицензионный ключ** — необходим для запуска бота
- **Ссылку на документацию** — пошаговая инструкция по установке

Вам потребуется:
1. Свой сервер (VPS/VDS) с Docker
2. Bot Token от [@BotFather](https://t.me/BotFather)
3. API ID и API Hash от [my.telegram.org](https://my.telegram.org)
4. Самостоятельно настроить и запустить Docker-контейнер

> Подробная инструкция — в разделе [SELF-HOST: установка](./self-host).

---

## HOSTING

**Мы запускаем бота за Вас — на нашем сервере.**

После оплаты:
1. Нажмите **"Настроить бота"**
2. Укажите Bot Token от [@BotFather](https://t.me/BotFather) (см. [Создание бота](./create-bot))
3. Пройдите **авторизацию** через веб-страницу (номер телефона + код из Telegram)
4. Бот **автоматически запустится** на нашем сервере

После запуска в меню **"Управление ботом"** доступны:
- **Запуск / Остановка / Перезапуск** бота
- **Просмотр логов** — последние 50 строк
- **Проверка статуса** — работает ли бот

> Подробная инструкция — в разделе [HOSTING: управление ботом](./hosting).

---

## HOSTING-PRO

**Выделенный VPS с максимальной производительностью.**

После оплаты:
1. Нажмите **"Настроить бота"**
2. Укажите Bot Token от [@BotFather](https://t.me/BotFather) (см. [Создание бота](./create-bot))
3. Пройдите **авторизацию** через веб-страницу
4. Администратор получит уведомление и выполнит **ручной деплой** на отдельном VPS
5. После развёртывания Вы получите уведомление — бот запущен

> Подробная инструкция — в разделе [HOSTING-PRO](./hosting-pro).

## Сравнение тарифов

| | SELF-HOST | HOSTING | HOSTING-PRO |
|---|:-:|:-:|:-:|
| Лицензионный ключ | ✅ | ✅ | ✅ |
| Настройка Bot Token | Самостоятельно | Через Service Bot | Через Service Bot |
| Авторизация | Самостоятельно | Через веб-страницу | Через веб-страницу |
| Где работает бот | Ваш сервер | Наш сервер (общий) | Выделенный VPS |
| Управление через бота | ❌ | ✅ | ✅ |
| Автоматический запуск | ❌ | ✅ | После деплоя админом |
| Мониторинг и авто-рестарт | ❌ | ✅ | ✅ |
